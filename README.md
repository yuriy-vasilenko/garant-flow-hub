# Гарант Маркет (garant-flow-hub)

Текущий режим проекта: локальный каталог в браузере (без сервера и без Supabase).

## Запуск

1. `npm install`
2. `npm run dev`

## Деплой на GitHub Pages

Пуш в ветку `main` запускает workflow `Deploy to GitHub Pages`.

Сайт публикуется по адресу:
`https://<ваш-логин>.github.io/garant-flow-hub/`

## Как работает каталог

- Категории и товары хранятся в `localStorage` этого браузера.
- Импорт/добавление/удаление работают локально.
- В другом браузере или на другом устройстве данных не будет (это ожидаемо для локального режима).

## Админка

Вход: логин `admin`, пароль `admin`.
